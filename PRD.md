# DevTools 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
DevTools 是一个面向开发者的在线工具集网站，提供常用的编码/解码、加密/哈希、时间转换等功能，帮助开发者提高日常工作效率。

### 1.2 目标用户
- 前端开发者
- 后端开发者
- 全栈开发者
- 测试工程师
- 运维工程师

### 1.3 核心价值
- **便捷性**: 无需安装，打开即用
- **隐私安全**: 纯前端计算，数据不离开浏览器
- **教育性**: 提供详细的概念介绍和多语言代码示例

## 2. 功能需求

### 2.1 Unix 时间戳工具

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 实时显示 | 显示当前时间戳（秒/毫秒）和本地/UTC 时间 | P0 |
| 时间戳转日期 | 将时间戳转换为本地时间和 UTC 时间 | P0 |
| 日期转时间戳 | 将日期时间转换为时间戳 | P0 |
| 自动检测单位 | 根据输入长度自动判断秒/毫秒 | P1 |
| 代码示例 | 提供 JS/Python/Java/Swift/Go 示例 | P1 |

### 2.2 Base64 编解码

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 文本编解码 | 支持 UTF-8 等字符编码 | P0 |
| 文件编解码 | 支持拖拽上传文件 | P0 |
| URL Safe 模式 | 支持 URL 安全的 Base64 | P1 |
| HEX 视图 | 展示解码后的二进制数据 | P1 |
| 代码示例 | 提供多语言示例 | P1 |

### 2.3 URL 编解码

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| encodeURIComponent | 编码 URL 参数值 | P0 |
| encodeURI | 编码完整 URL | P0 |
| escape | 兼容旧代码（标注已废弃） | P2 |
| 模式对比 | 对比三种模式的编码结果 | P1 |
| 保留字符表 | 展示各模式不编码的字符 | P1 |

### 2.4 Hash 计算器

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 文本 Hash | 计算文本的哈希值 | P0 |
| 文件 Hash | 计算文件的哈希值（最大 50MB） | P0 |
| 多算法支持 | MD5/SHA1/SHA256/SHA384/SHA512/SHA3 | P0 |
| 输出格式 | 支持 Hex（大小写）和 Base64 | P1 |
| HEX 视图 | 展示文件二进制内容 | P2 |

### 2.5 UUID 生成器

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| v4 随机生成 | 使用 crypto.getRandomValues | P0 |
| v3 MD5 生成 | 基于命名空间和名称 | P1 |
| v5 SHA-1 生成 | 基于命名空间和名称 | P1 |
| 批量生成 | 一次生成多个 UUID | P1 |
| 格式选项 | 标准/无连字符/大括号/URN | P1 |
| UUID 解析 | 解析 UUID 的版本和变体 | P2 |

## 3. 非功能需求

### 3.1 性能
- 页面首次加载 < 1s（3G 网络）
- 工具响应时间 < 100ms
- 大文件处理有进度提示

### 3.2 兼容性
- 支持主流现代浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，支持移动端访问

### 3.3 安全性
- 所有计算在浏览器本地完成
- 不收集任何用户数据
- 不使用第三方追踪脚本

### 3.4 可访问性
- 支持键盘导航
- 支持屏幕阅读器
- 符合 WCAG 2.1 AA 标准

## 4. 用户体验

### 4.1 主题
- 默认跟随系统偏好
- 支持手动切换亮/暗主题
- 主题偏好保存到 localStorage

### 4.2 交互
- 一键复制结果
- 实时预览（如时间戳）
- 清晰的错误提示

### 4.3 教育性
- 每个工具提供概念介绍
- 提供多语言代码示例
- 展示算法对比和选择建议

## 5. 限制条件

| 限制项 | 说明 |
|--------|------|
| 最大文件大小 | 50MB |
| 批量生成上限 | 100 个 |
| 不支持功能 | UUID v1（需要 MAC 地址） |
